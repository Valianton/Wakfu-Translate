### Инструкция: как вынести новые строки и добавить их в `Translated\Texts\text_en.properties`

Запустить файл compare.py через PowerShall либо командную строку командой:
```
python compare2.py
```

1. После исполнения скрипта создастся файл `texts_en.properties`, который необходимо поместить в папке `
Translated/Texts`. Этот файл уже содержит новые добавленные строки, а также в нем удалены старые строки


2. После исполнения скрипта создастся файл `missing.properties`, который содержит новые добавленные строки
